About Model Canvas

Model Canvas is an interactive machine learning visualization platform that helps you understand how different algorithms work through real-time decision boundary visualization.

üéØ Key Features

  • Real-time Training Visualization - Watch models learn and see decision boundaries form
  • Multiple Algorithms - Logistic Regression, KNN, SVM, Decision Trees, Random Forest, Neural Networks
  • Diverse Datasets - Iris, Wine, Breast Cancer, Diabetes, and synthetic datasets
  • Dual Training Modes - Educational (step-by-step) vs Optimized (full speed)
  • Performance Metrics - Live accuracy, loss tracking, and confusion matrices
  • Feature Visualization - Switch between raw features and PCA projections

üí° How to Use

  1. Select a dataset and algorithm from the top controls
  2. Adjust hyperparameters as they appear
  3. Click "Train Model" to start the visualization
  4. Monitor metrics in the right panel
  5. Use "Educational" mode to see step-by-step learning
